Claude Code Workflow Studio：可视化工作流编辑器



还在为手写Claude Code复杂工作流熬到凌晨？GitHub上2.3k星的这款开源工具直接把流程设计变成"搭积木"——ClaudeCode Workflow Studio，专为Claude Code、VS Code重度用户量身打造的可视化工作流编辑器，堪称开发生产力外挂！


无需手写冗长代码或配置文件，只需拖拽左侧节点库中的提示词、子智能体、人工确认、条件判断等组件，就能像画流程图一样搭建Claude Code自动化任务、技能（Skills）乃至复杂Agent逻辑。有意思的是，除需联网调用的MCP工具外，所有编辑与生成逻辑均在VS Code本地运行，隐私配置全程不碰云端，安全感直接拉满。界面还会自动匹配VS Code的语言设置，原生支持中文，完全没有语言壁垒。


嫌拖拽节点太繁琐？直接点击「Edit with AI」，用大白话描述你的工作流需求，AI就能自动帮你把节点摆放、连线全部搞定——比如只需一句"用GitHub MCP工具获取PR详情，按PR规模路由到小/中/大型代码审查流程，发现问题就通知用户，审查通过就请求批准权限"，分分钟生成可直接运行的自动化代码审查工作流，这效率简直是降维打击。

更贴心的是，ClaudeCode Workflow Studio还打通了MCP与Skills生态：既能通过MCP直接调用外部工具，也能复用已有的自定义Skill，异构工具与存量能力无缝衔接。

还支持插入「询问用户」节点，AI完成任务后会弹出选项让你选择通过或打回，人机协作边界清晰可控。画布上的工作流设计完成后，点击导出就能自动生成Claude Code可直接识别的.claude/agents/与.claude/commands/文件，无需手动格式转换，终端敲命令就能直接运行，一站式自动化到底。

上手门槛极低，只需两步就能启动：

首先确保本地已安装Claude Code命令行工具，未安装的话在终端执行：

npm install -g @anthropic-ai/claude-code
然后打开VS Code，在扩展市场搜索「Claude Code Workflow Studio」完成安装。

图片
按下Ctrl+Shift+P打开命令面板，输入「Claude Code」，选择「Claude Code Workflow Studio: Open Editor」，就能看到空白画布工作台，开始你的可视化工作流搭建之旅。
